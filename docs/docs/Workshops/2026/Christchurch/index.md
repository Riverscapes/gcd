---
title: GCD Workshop at IAG
---

A one day introduction to the GCD software covering the following topics:

- Topographic differencing - a recap
- Managing DEM uncertainty
- GCD tools: 
- Standalone and ArcGIS Pro Add-In
- Ashley River datasets
- Building a GCD project 
- Creating simple error models
- Interpreting DEM difference results
- Comparing results
- Geomorphic interpretation

## Location

Hosted at the University of Canterbury as part of the [11th IAG Conference](https://www.confer.co.nz/icg2026/), Christchurch, New Zealand.
