This privacy policy governs your use of the software application GCD (“Application”) created by North Arrow Research Ltd.

## 1. Personal Information

The Application does not collect, store, or transmit any personally identifiable information (PII). We do not ask for, nor do we have access to, your name, email address, physical address, or any other private data.

## 2. Anonymous Telemetry

To help us improve the Application’s performance and stability, the Application may collect anonymous usage events (e.g., app launch, feature usage, or error logs).

This data is completely anonymous and cannot be used to identify you or your specific device.

This feature is strictly optional.

## 3. User Choice and Opt-Out

You have full control over the telemetry feature. You can opt-out of anonymous data collection at any time by navigating to the Options form within the Application and unchecking the telemetry setting.

## 4. Data Access and Permissions

The Application requires certain system permissions (such as "runFullTrust") to operate as a desktop application on Windows. These permissions are used solely for the local execution of the Application’s features and do not imply any access to your personal data.

If you have any questions regarding privacy while using the Application, please contact us at support@riverscapes.freshdesk.com
